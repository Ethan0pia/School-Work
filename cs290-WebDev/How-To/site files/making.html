How-To make a Blackjack game using Deck of Cards API

Now, for the fun part...

Planning:

First, let's think about what we are trying to accomplish for a minute. To play Blackjack, we need to create a deck, draw cards, and maybe reshuffle our decks. Do we need a discard pile? A custom deck? Not unless we are planning on changing the Blackjack rules, which we are not. Usually, casinos play with 6-8 decks, so let's plan on using 7 decks. We will also need a way to deal, hit, and stand; the 3 most important parts of Blackjack. A way of keeping score, keeping count of card values, and a way to show who won will also be needed.

Step 2-The CSS:

So, we have our website completed, but it doesn't look very pretty, does it? So, what do we need?

  1. A cool background and colored text
  2. larger and colorful buttons
  3. increase the size of the div for images
  4. set the size of the card images we will be adding

The first step in making a CSS file is to create a new text document and name it "Blackjack.css". Remember how we added that name in our link of the HTML page? Now it comes into play.

Let's start with fixing the background. Go onto google images and find a nice background picture. I like green felt, as you can tell by the blackground picture of this how-to guide. Once you have found an image, download it to your computer and put it in the same folder as your Blackjack.html file.

Inside your CSS file, type in "body" followed by an opening and closing squiggly brackets. Inside those brackets, type "background-image: url("felt.jpg");". background-image: changes to background of body. url("felt.jpg") changes the image to the location inside the parenthesis. Replace "felt.jpg" with the name of your image. Underneath the background-image, add "color:yellow;". Color: changes the text color of all the text inside the body to yellow. I find yellow easier to read with a green background. After doing this and opening your HTML page, you should have a cool background and yellow text.

The next step is to create larger and colorful buttons. First, we need to adjust the wrapper div that contains the buttons. To adjust the div with the id of "wrapper", we need to type in #wrapper. Do this on the line under the closing squiggly bracket of the body. Then, add opening and closing squigglies next to #wrapper. Inside those squigglies, let's set the "overflow: normal;" to stop the score from being displayed next to the wrapper.

Once the wrapper has been adjusted, create 3 more sections called #first, #second, and #third. Set them all to have "float:left;" and "width: 100px;". Your CSS file should look like this:

We should now add some color and size to the buttons to make them easier to click and stand out more. We have 3 buttons, so we need to create #deal, #hit, and #stand sections in our CSS. Set all 3 to "width:80px" and "height:50px". This will set their width and height to a set 80 pixels and 50 pixels. Also, let's add a "background-color:" to each of the buttons. For deal, let's set it to "background-color:red;", hit to "cornflowerblue;", and stand to "yellowgreen;". Also, black lettering is hard to see on a red background, so let's change the color of text for deal to white with "color:white;". Once you have completed this, your HTML page should look like this when opened:

Finally, we need to adjust the size of the card area. We don't have much space between "Dealer" and "Dealer Total" or "Player" and "Player Total". To fix this, we will do several things. The first thing we need to do is create #dealer and #player fields in our CSS file. Inside them, we need to set "height: 175px;" because 175px is a good height for a playing card on an average computer screen. The second step is to create am "img" field in the css. Since it is not an ID, it will change all img on the page, even img that we add later. Since we decided that 175px is a good height for a card, put that inside the img field as well. Also, with a height of 175px, 125px is a good width. So, add "width:125px;" to the img feild. Once complete, your HTML page should look like this:
